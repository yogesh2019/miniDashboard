Welcome to the Dashboard Demo

This is a demo application showcasing a dashboard integrated with an AI-powered assistant. The assistant answers questions using real dashboard data in a safe and controlled way.

What you can do here

  • View dashboard records loaded from a live backend
  • Ask questions using the AI chatbot
  • See how Retrieval-Augmented Generation (RAG) works with real data
